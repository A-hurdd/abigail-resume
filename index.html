What I Build

Personal Project
Off-Grid Mobile Terminal
Termux / Android / Local Inference / SQLite

Converted a spare phone into a fully offline reference and problem-solving terminal using Termux. Runs local inference with a custom database of information. Built for scenarios with no internet access.

The problem: in field environments, internet isn't guaranteed. I wanted a device that could answer questions, run lookups, and serve as a technical reference without any connection.

Built on a spare Android phone using Termux as the shell environment. Integrated a lightweight local inference model with a custom SQLite database of reference material — equipment specs, protocols, and quick-reference data.

Designed to be portable, low-power, and fully self-contained. No cloud dependency, no setup required in the field.
